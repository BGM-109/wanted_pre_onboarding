# TF
term frequency 단어빈도 
문서 내에 얼마나 자주 등장하는지를 나타내는 값. 높을 수록 문서에서 중요하다고 생각할수있다.

# DF
document frequency 문서 빈도

# IDF
inverse document frequency 역 문서 빈도

# TF-IDF
TF와 IDF를 곱한 값으로 점수가 높은 단어일 수록 다른 문서에는 많지 않고 해당 문서에서 자주 등장하는 단어를 의미한다.
